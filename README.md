# نظام إدارة المستفيدين - دعوة الحق

نظام متكامل لإدارة المستفيدين والمبادرات الخيرية مع نظام كشفات شهرية وتصدير PDF.

## 🌟 الميزات الجديدة

### نظام الكشفات الشهرية
- إنشاء كشفات شهرية للمستفيدين
- تحديد مبلغ شهري لكل مستفيد
- حساب إجمالي الكشف تلقائياً
- تصفية الكشفات حسب الشهر والسنة
- عرض إحصائيات شهرية وسنوية

### تصدير PDF
- تصدير كشف شهري إلى PDF
- تصدير كشف مفصل مع جميع البيانات
- تصدير تقرير شهري شامل
- تصدير تقرير سنوي
- دعم الخط العربي في PDF

### إدارة المبالغ
- تحديد مبلغ شهري لكل مستفيد
- تعديل المبالغ بسهولة
- عرض إجمالي المبالغ في الكشف
- حساب إجمالي الشهر والسنة

## 🚀 الميزات الأساسية

### إدارة المستفيدين
- إضافة وتعديل وحذف المستفيدين
- رفع صور الهوية للمتزوجين
- تحديد الأولوية حسب الدخل وحجم الأسرة
- البحث والتصفية المتقدمة

### إدارة المبادرات
- إنشاء مبادرات خيرية
- ربط المستفيدين بالمبادرات
- تتبع الميزانيات والإنفاق

### إدارة المتطوعين
- تسجيل المتطوعين الجدد
- إدارة بيانات المتطوعين
- تتبع نشاطات التطوع

### النظام المالي
- تتبع الإيرادات والمصروفات
- إدارة الميزانيات
- تقارير مالية مفصلة

### الأمان والصلاحيات
- نظام تسجيل دخول آمن
- إدارة الأدوار والصلاحيات
- حماية البيانات الحساسة

## 🛠️ التقنيات المستخدمة

- **Frontend**: React.js, Redux Toolkit, Framer Motion
- **Styling**: CSS3 مع متغيرات مخصصة
- **PDF**: jsPDF مع دعم الخط العربي
- **Storage**: IndexedDB و localStorage
- **Deployment**: Vercel مع ربط الدومين

## 📦 التثبيت والتشغيل

### المتطلبات
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

### خطوات التثبيت

1. استنساخ المشروع:
```bash
git clone https://github.com/yourusername/dawaaelhaq-website.git
cd dawaaelhaq-website
```

2. تثبيت التبعيات:
```bash
npm install
```

3. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

4. بناء المشروع للإنتاج:
```bash
npm run build
```

## 🌐 النشر على الإنترنت

### ربط الدومين المخصص
تم ربط الموقع بالدومين `dawaaelhaq.top` من خلال:

1. **Vercel Deployment**:
   - رفع المشروع على Vercel
   - ربط الدومين المخصص
   - تكوين SSL تلقائي

2. **DNS Configuration**:
   - إعداد سجلات DNS في لوحة تحكم الدومين
   - توجيه الدومين إلى Vercel

### GitHub Pages (بديل)
```bash
npm run deploy:github
```

## 📊 استخدام النظام

### إنشاء كشف شهري
1. انتقل إلى صفحة "إدارة الكشفات"
2. اضغط على "إنشاء كشف جديد"
3. أدخل اسم الكشف ووصفه
4. أضف المستفيدين مع تحديد المبلغ الشهري لكل منهم

### تصدير PDF
1. في صفحة الكشفات، اضغط على أيقونة PDF
2. اختر نوع التصدير:
   - كشف عادي
   - كشف مفصل
   - تقرير شهري
   - تقرير سنوي

### إدارة المبالغ
1. في قائمة المستفيدين في الكشف
2. اضغط على أيقونة المال لتعديل المبلغ
3. أدخل المبلغ الجديد وحفظ
